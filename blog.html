Important FAQ!

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web document as a tree of objects, allowing scripts to manipulate the content and structure of web pages

How do you select an element from the DOM?

You can select an element from the DOM using methods like `document.getElementById()`.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element to manage events for multiple child elements, improving performance and simplifying code by reducing the number of listeners.

How do you manipulate on element's attributes and styles using the DOM?

You can manipulate an element's attributes using the `setAttribute()` and `getAttribute()` methods, and modify its styles by changing the `style` property directly (e.g., `element.style.color = "red"`).
